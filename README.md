# Cinetopia

## Descrição
Primeiro projeto utilizando UIKit com ViewCode para construção das telas.

O aplicativo consiste básicamente no consumo de uma API na qual é disponibilizada uma lista de filmes. 

A tela principal do app exibe uma lista dos titulos, sendo exibido o nome do filme, poster e data de lançamento. Nessa tela é possivel marcar o filme como favorito ou acessar a tela de detalhes do filme, onde é exibida a sinopse do filme.

## Description
First project using UIKit with ViewCode for screen construction.

The application basically consists of consuming an API that provides a list of movies.

The app's main screen displays a list of titles, showing the movie name, poster, and release date. On this screen, it's possible to mark a movie as a favorite or access the movie details screen, where the movie synopsis is displayed.
